# KAZ.DB
Basic Database Module

**Define** =>
```js
const Kaz = require("kaz.db"); 
const db = new Kaz(); //|| File Patch
```
**Download Module**
```js
npm install kaz.db
``` 

# Examples
```js
db.set("Money", "500");
db.get("Money"); 
db.delete("Money");
db.has("Money"); 
db.fetch("Money");
db.add("Money", 100); 
db.subtract("mONEY", 5); 
db.set("array", [ "apple" ]);
db.push("array", "orange"); 
db.clear(); 
db.console("Connect Kaz.db Thanks For Using")
```
